Inclusion of CSS

Internal CSS

OR

Embedded CSS

We can put our CSS rules into an HTML document using the <style> element.

This <style> element is placed inside the <head> tags.

Rules defined using this <style> element inside the <head> element will be applied to all the elements, and also to all the instances of the specified element available in the web document.

Inline CSS

We can use the style attribute of any HTML element to define style rules. These rules will be applied to a particular element only.
